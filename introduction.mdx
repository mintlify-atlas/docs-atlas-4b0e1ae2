Yoast SEO is a WordPress plugin that helps you optimize your site's content and technical setup for search engines. This page covers what Yoast SEO does, its key features, system requirements, and the difference between the free and premium editions.

## What is Yoast SEO?

Yoast SEO adds an analysis panel to every post, page, and custom post type in the WordPress editor. As you write, the plugin evaluates your content against SEO best practices and readability guidelines, then surfaces actionable feedback through color-coded bullet indicators (green, orange, and red). It also generates the meta tags, XML sitemaps, and structured data (schema.org) that search engines use to understand and index your site.

The plugin ships with a first-time configuration wizard that walks you through the most impactful settings — site representation, social profiles, and the indexing of your content — so you can get a solid baseline in place without touching PHP or editing `wp-config.php`.

## System requirements

| Requirement | Minimum version |
|---|---|
| WordPress | 6.8 |
| PHP | 7.4 |

These values come from the `YOAST_SEO_WP_REQUIRED` and `YOAST_SEO_PHP_REQUIRED` constants defined in `wp-seo-main.php`. Yoast tests against the latest stable WordPress release; running an older version may work but is unsupported.

## Free vs. premium

The free edition, available on [WordPress.org](https://wordpress.org/plugins/wordpress-seo/), covers the full SEO and readability analysis, XML sitemaps, schema output, and the first-time configuration wizard. Most sites can run entirely on the free edition.

Yoast SEO Premium adds:

- **Multiple focus keyphrases** — optimize a single piece of content for up to five related keyphrases.
- **Internal linking suggestions** — the plugin recommends existing posts to link to while you write.
- **Redirect manager** — create and manage 301, 302, and other redirects from inside WordPress.
- **AI-generated title and description suggestions** — draft meta copy from within the editor.
- **24/7 support** — email and chat support from the Yoast team.
